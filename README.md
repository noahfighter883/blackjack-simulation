# Blackjack Console Simulation (Java)

Console-based blackjack game supporting splits, doubles, and bankroll tracking.

## Features

- Hit / Stand
- Double Down
- Splits
- Bankroll management
- Dealer logic implementation

## Design

Object-oriented implementation with:
- Card class
- Deck class
- Player class
- Game controller logic

## Compile

```bash
javac Blackjack.java
